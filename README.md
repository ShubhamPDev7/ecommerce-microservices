# E-Commerce Microservices

A learning-focused microservices project built with Spring Boot and Spring Cloud. The project demonstrates service decomposition, service discovery, inter-service communication, and modern backend development practices.

## Architecture

### Services

* Discovery Service (Netflix Eureka Server)
* Inventory Service
* Order Service

## Tech Stack

* Java 21
* Spring Boot
* Spring Data JPA
* Spring Cloud Netflix Eureka
* PostgreSQL
* Maven
* Lombok

## Features Implemented

### Inventory Service

* Product management
* Product inventory tracking
* PostgreSQL integration
* REST APIs

### Order Service

* Order creation
* Order item management
* Order status tracking
* PostgreSQL integration
* REST APIs

### Service Discovery

* Eureka Server setup
* Service registration
* Service discovery using Eureka Client

## Project Structure
```text
 eCommerce/
├── discovery-service
├── inventory-service
└── order-service
```
